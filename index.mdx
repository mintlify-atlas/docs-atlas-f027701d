## What is cineCDMX?

cineCDMX is a full-stack Django web application that powers cinema operations for Mexico City locations. It provides a public-facing movie catalog with ticket purchasing, an employee dashboard for managing movies and screenings, and an admin panel for branch and user management.

## Key capabilities

- **Multi-branch operations** — Manage screenings independently across CDMX Norte, Oriente, Reforma, Perisur and more
- **TMDB integration** — Automatically pull movie posters, synopses, and runtime from The Movie Database API
- **Role-based access** — Three distinct roles: Cliente (customer), Empleado (employee), and Admin
- **Google OAuth** — Customers can sign in with their Google account via django-allauth
- **Conflict detection** — The system prevents double-booking a sala at the same date and time
- **Interactive seat map** — Customers choose specific seats before purchase; occupied seats are shown in real time
